Spiel the Ship wird in AR nachgebaut

- Jeder Mitspieler hat Marker mit Sicherheitsnadel am Tshirt und repräsentiert eine der Rollen / Passagiere
- Jede Runde werden die Rollen / Passagiere neu verteilt und man muss Marker erst scannen um festzustellen ob target oder nicht
	-> Spieler kann überlegen, ob er direkt angreift oder erst mal so tut und weiter geht
	
- Mögliche Erweiterung (GPS? Future Work?)
	- no kill zones
	- security
	- In GUI wird angezeigt ob gerade unter Beobachtung oder nicht

- auf Marker wird Model des Passagiers / der Rolle gerendert (3D Model oder für erste Version 2D Bild)
- Waffen sollten auch als 3D Model gerendert werden
	
- Man kann Waffen einsammeln (Marker die in der Gegend hängen - Marker sind dann für diese Runde aufgebraucht - Waffen können also nur einmal eingesammelt werden)
- Um sein Ziel ausuzschalten, dessen Marker anvisieren und Angriff Button drücken - je nach Waffe mehr oder weniger schaden
- Server verteilt Info über Ziele der Runde also wen man tot und wer lebt und tot ist
- Alle infos auf GUI: Ziel, Waffe, Leben, Punkte

- Besondere Waffen geben mehr Punkte (Waffe der Runde - Achievements etc..)

- Man muss die Waffe erst ziehen, der andere bekommt das angezeigt auf der GUI

- Es gibt auch Punkte, wenn man denjenigen tötet, der einen selbst als Ziel hat
